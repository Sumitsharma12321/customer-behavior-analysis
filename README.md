## 📖 Overview

This project performs a complete customer behavior analysis on a retail shopping dataset. The goal is to understand how customers shop — their spending habits, product preferences, seasonal trends, and subscription behavior — and present the findings through an interactive Power BI dashboard.

---

## 📂 Dataset

- **File:** `customer_shopping_behavior.csv`
- **Records:** 3,900 rows × 18 columns
- **Age Range:** 18 – 70 years
- **Locations:** 50 US states

### Columns

| Column | Description |
|--------|-------------|
| `Customer ID` | Unique customer identifier |
| `Age` | Customer age |
| `Gender` | Male / Female |
| `Item Purchased` | Product name |
| `Category` | Clothing, Accessories, Footwear, Outerwear |
| `Purchase Amount (USD)` | Transaction value |
| `Location` | US state |
| `Size` | Product size |
| `Color` | Product color |
| `Season` | Season of purchase |
| `Review Rating` | Rating (1–5) |
| `Subscription Status` | Yes / No |
| `Shipping Type` | Express, Standard, Free Shipping, etc. |
| `Discount Applied` | Yes / No |
| `Promo Code Used` | Yes / No |
| `Previous Purchases` | Number of past purchases |
| `Payment Method` | PayPal, Credit Card, Cash, etc. |
| `Frequency of Purchases` | Weekly, Monthly, Fortnightly, etc. |

## 🐍 Python Analysis

The Jupyter Notebook (`Customer_behavior_analysis_using_Python.ipynb`) covers:

**1. Data Loading & Exploration**
- Loaded dataset using Pandas
- Explored structure using `df.head()`, `df.info()`, `df.describe()`

**2. Data Cleaning**
- Checked null values — `Review Rating` had missing data
- Imputed missing `Review Rating` with **median rating per category**
- Renamed all columns to `snake_case` for consistency
- Dropped `promo_code_used` column (100% identical to `discount_applied`)

**3. Feature Engineering**
- Created `age_group` using `pd.qcut()` → 4 groups: Young Adult, Adult, Middle-aged, Senior
- Created `purchase_frequency_days` by mapping frequency labels to numeric days

**4. Load to MySQL**
- Connected to MySQL using `SQLAlchemy`
- Exported cleaned DataFrame to `customer` table in `mydata` database

## 🗄️ SQL Analysis

10 business questions answered using MySQL:

| # | Question |
|---|----------|
| Q1 | Total revenue generated by Male vs Female customers |
| Q2 | Customers who used discount but still spent above average |
| Q3 | Top 5 products with highest average review rating |
| Q4 | Average purchase amount — Standard vs Express shipping |
| Q5 | Do subscribed customers spend more? (avg spend + total revenue) |
| Q6 | Top 5 products with highest discount usage rate |
| Q7 | Customer segmentation — New, Returning, Loyal |
| Q8 | Top 3 most purchased products within each category |
| Q9 | Are repeat buyers (5+ purchases) more likely to subscribe? |
| Q10 | Revenue contribution by age group |

**SQL concepts used:** `GROUP BY`, `HAVING`, `Subqueries`, `CTEs (WITH clause)`, `Window Functions (ROW_NUMBER)`, `CASE WHEN`, `ROUND`, `Aggregate Functions`

## 💡 Key Insights

- **Total Revenue:** $233,081 from 3,900 customers
- **Average Purchase Value:** $59.76
- **Clothing** is the top revenue category — **$104,264 (44.7% of total revenue)**
- **Male customers** contribute **67.7% of revenue** ($157,890) vs Female ($75,191)
- **Loyal customers** (10+ purchases) make up **79.9% of all customers** (3,116 out of 3,900)
- Only **27% customers are subscribed** (1,053 out of 3,900)
- **51+ age group** is the highest revenue contributor — **$88,480**
- Sales are nearly **evenly distributed across all 4 seasons**
- **PayPal** is the most used payment method (677 customers)
- Average review rating across all products: **3.75 / 5**
