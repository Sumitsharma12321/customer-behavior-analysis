# 🛍️ Customer Shopping Behavior Analysis – Data Analytics Project
📌 Overview

This project analyzes customer shopping behavior using a structured data analytics workflow. The goal is to identify purchasing patterns, customer segments, and revenue drivers that can help businesses make better strategic decisions.

The project follows a complete end-to-end analytics pipeline, including:
Data loading and preprocessing in Python
Exploratory Data Analysis (EDA)
Data cleaning and feature engineering
Business analysis using SQL
Interactive dashboard creation in Power BI
Report and presentation generation

Dataset used: customer_shopping_behavior.csv

📂 Dataset

Dataset Name: Customer Shopping Behavior Dataset
Source: Kaggle

Dataset contains 3,900 customer purchase records and 18 features including demographics, purchasing behavior, and product information.

4️⃣ SQL Analysis (MySQL)

The cleaned dataset was imported into MySQL for business analysis.

Key questions answered:

💰 Revenue generated by Male vs Female customers

🏷 Customers using discounts but spending above average

⭐ Top 5 products with highest review ratings

🚚 Purchase comparison between Standard vs Express shipping

💳 Spending comparison of Subscribers vs Non-Subscribers

🛍 Products with highest discount usage rate

👥 Customer segmentation (New / Returning / Loyal)

🏆 Top 3 products in each category

🔁 Relationship between repeat buyers and subscriptions

👨‍👩‍👧 Revenue contribution by age group

Example SQL query used:

SELECT gender, SUM(purchase_amount) AS total_revenue
FROM customer
GROUP BY gender;

📈 Key Insights / Results

Important insights discovered from the analysis:

Certain product categories contribute majority of revenue

Customers using discounts often still spend above average

Express shipping users tend to spend more

Loyal customers generate a large portion of revenue

Some products show very high discount dependency

These insights can help businesses improve marketing strategy, pricing, and customer retention programs.
